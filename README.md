# 💓 Heart Disease Prediction System

An advanced Machine Learning-based system to predict the likelihood of heart disease using patient health data. The project integrates multiple ML models and provides real-time predictions through an interactive web application.

---

## 🚀 Features

- Predicts heart disease risk based on medical parameters
- Uses multiple ML models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
- Evaluates models using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - ROC-AUC
- Displays feature importance for interpretability
- Provides risk probability (%) instead of just binary output
- Interactive web interface using Streamlit

---

## 🧠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## 📊 Dataset

- UCI Heart Disease Dataset (Cleveland dataset)
- Includes features like:
  - Age
  - Sex
  - Chest Pain Type
  - Blood Pressure
  - Cholesterol
  - ECG results
  - Maximum Heart Rate
